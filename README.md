# OKCupid: Date-A-Scientist

## 1. Project Goals
* **Primary Objective:** Classify whether a user is in a STEM field (Science, Technology, Engineering, Math) based on profile data.
* **Secondary Objective:** Identify lifestyle markers (diet, drinking habits, education) correlated with scientific professions.

## 2. Data
* **Source:** Anonymized OKCupid dataset (~60,000 entries).
* **Key Features:** Age, status, sex, orientation, diet, drinks, drugs, education, job, religion.

## 3. Analysis
* **EDA:** Visualizing distributions using Matplotlib and Seaborn.
* **Modeling:** Comparing K-Nearest Neighbors (K-NN), Multinomial Naive Bayes, and Support Vector Machines (SVM).

## 4. Evaluation
* **Metrics:** Accuracy, Precision, Recall, and Confusion Matrix.
* **Validation:** 80/20 Train-Test split.

## 5. Output
* A predictive model and a statistical profile of the "Average Scientist" vs. the general user.
